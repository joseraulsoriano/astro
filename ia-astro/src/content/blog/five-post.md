**Tech Finder** es un ejercicio de **descubrimiento de herramientas** con énfasis en filtros útiles y estados de carga honestos. El núcleo vive en **React** y **TypeScript** para mantener interfaces predecibles a medida que crecen las fuentes.

**Python** aparece donde hace falta normalizar datos o exponer endpoints sencillos sin acoplar el front al detalle del origen. **HTML** semántico y accesibilidad básica se cuidaron desde las plantillas para no depender solo de helpers del framework.

Como mejoras continuas valdrían cache en cliente, modo degradado cuando la API falla y pruebas de componentes para las vistas de resultado.
